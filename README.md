# dbt-editor

`dbt-editor` is a locally hosted web app for working with dbt projects on a developer machine.

## Goals

- Browse local dbt project files and folders.
- Open and edit dbt models, YAML, SQL, and supporting project files.
- Run selected CLI commands from the UI for local workflows.
- Keep the tool lightweight and focused on local development.

## Initial product direction

The first version of the app will act as a file explorer and editor for dbt repositories, with an interface to run common local CLI commands. It is intended for local use on a trusted machine rather than multi-user deployment.
